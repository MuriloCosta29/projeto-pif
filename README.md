# Stan James

Jogo de cobrança de faltas desenvolvido em C com raylib para a disciplina de Programação Imperativa e Funcional.

## Sobre o jogo

O objetivo do jogo é marcar a maior quantidade de gols possível em cobranças de falta. O jogador usa a barra de espaço para definir os atributos do chute, como direção, altura, curva e potência.

A partida termina quando o jogador esgota suas chances.

## Tecnologias

- Linguagem C
- raylib
- Makefile
